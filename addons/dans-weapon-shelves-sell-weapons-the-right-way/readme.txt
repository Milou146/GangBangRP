PERMANENT SHELVES TUTORIAL:

As admin spawn in a shelf and edit the shelf to your liking.
Once the shelf is ready to be saved use the command "!permshelf"
!perm shelf will registered the shelf as permanent
Continue adding more permanent shelves and using the "!permshelf" command
If you make a mistake while editting, fix it then use the "!permshelf" command again to adjust the mistakes.
Once all shelves are placed and mistakes are cleared, use the command "!saveshelves" to successfully create a datafile with all the shelf data.
If anything goes wrong in this process use the command "!clearshelfdata" and restart.
